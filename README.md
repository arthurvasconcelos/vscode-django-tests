# VSCode Django Tests Provider

Django tests using VSCode's test explorer

## Sample Setup

```sh
pyenv virtualenv 3.12.2 vscode-django-tests

pyenv activate vscode-django-tests

cd sample

source ./env.sh

pip install
```
